# TempForge
TempForge — это коллекция из семи независимых реализаций конвертера температур, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода к одной задаче, а также для предоставления удобного инструмента для преобразования между различными температурными шкалами с расширенными возможностями
Уникальные возможности (общие для всех версий)
🌡️ Поддержка множества шкал: Цельсий (°C), Фаренгейт (°F), Кельвин (K), Ранкин (°Ra), Реомюр (°Ré), Ньютон (°N), Делиль (°De), Рёмер (°Rø)

🔄 Гибкое преобразование: конвертация между любыми двумя шкалами, а также пакетное преобразование списка значений

📊 Визуализация: построение таблицы эквивалентов для диапазона температур (в некоторых версиях)

🔁 Обратный конвертер: автоматический расчёт обратного преобразования

📝 Экспорт результатов в текстовый файл, CSV или JSON

🖥️ Интерфейсы:

Командная строка (CLI) с гибкими опциями

Графический интерфейс (GUI) в Python, Java и C#

Веб-интерфейс в JavaScript и PHP

🧮 Точность: использование чисел с плавающей запятой двойной точности, округление до заданного количества знаков

⚡ Пакетный режим: преобразование нескольких значений из файла или ввода

🚀 Быстрый старт
Общее для всех версий
Все версии поддерживают следующие шкалы (сокращения для CLI):

c — Цельсий

f — Фаренгейт

k — Кельвин

ra — Ранкин

re — Реомюр

n — Ньютон

de — Делиль

ro — Рёмер

Python
bash
python temp_convert.py --value 100 --from c --to f
Для GUI:

bash
python temp_convert.py --gui
JavaScript (Node.js)
bash
node temp_convert.js --value 100 --from c --to f
Go
bash
go run temp_convert.go -value 100 -from c -to f
Rust
bash
cargo run -- --value 100 --from c --to f
Java
bash
javac TempConverter.java && java TempConverter
# Затем следуйте интерактивному меню.
C#
bash
csc /reference:System.Windows.Forms.dll /reference:System.Drawing.dll TempConverter.cs
TempConverter.exe
PHP (веб)
bash
php -S localhost:8000
# Откройте http://localhost:8000/temp_convert.php
Для CLI:

bash
php temp_convert.php --value=100 --from=c --to=f
🧪 Примеры использования
Конвертировать 100°C в °F:
bash
python temp_convert.py --value 100 --from c --to f
Вывод:

text
100.00 °C = 212.00 °F
Пакетное конвертирование из файла:
bash
python temp_convert.py --batch input.txt --from c --to f --output results.csv
Получить таблицу эквивалентов для диапазона:
bash
python temp_convert.py --range 0 100 --step 10 --from c --to f
🛠️ Расширение и кастомизация
В каждой версии вы можете легко добавить новые шкалы, изменить точность вывода, добавить поддержку новых форматов экспорта. Подробные инструкции приведены в комментариях внутри кода.
